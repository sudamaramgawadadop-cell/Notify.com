HTML Introduction

HTML (HyperText Markup Language) is the standard language used to create web pages. 

<!DOCTYPE html>
<html>
<head>
<title>My First Page</title>
</head>

<body>
Hello World
</body>

</html>

HTML Elements

An HTML element usually consists of an opening tag, content and a closing tag. 

<h1>Heading</h1>

<p>Paragraph</p>

HTML Headings

HTML has six heading tags from h1 to h6. 

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<h3>Heading 3</h3>

Paragraph

Paragraphs are written using the p tag. 

<p>
This is a paragraph.
</p>

Links

Links are created using the anchor tag. 

<a href="https://google.com">
Visit Google
</a>

Images

<img src="image.jpg"
alt="Example">

HTML Lists

Lists are used to group related items.

Unordered List

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript

HTML Tables

Tables are used to display data in rows and columns.

<table border="1">
<tr>
<th>Name</th>
<th>Age</th>
</tr>

<tr>
<td>Rahul</td>
<td>20</td>
</tr>

</table>
Name Age
Rahul 20

HTML Forms

Forms collect user input.

<form>

<input
type="text"
placeholder="Name">

<br><br>

<input
type="email"
placeholder="Email">

<br><br>

<button>
Submit
</button>

</form>




Semantic HTML

Semantic tags describe the meaning of the content. 

<header></header>

<nav></nav>

<main></main>

<section></section>

<article></article>

<footer></footer>

CSS Introduction

CSS stands for Cascading Style Sheets. It is used to style HTML pages. 

h1{
color:blue;
}

Hello Xstar

CSS Selectors

Selectors target HTML elements. 

p{
color:red;
}

This is a red paragraph.

CSS Colors

You can use names, HEX, RGB and HSL colors. 

body{
background:#f4f4f4;
}

h1{
color:#0d6efd;
}

CSS Background

body{
background-color:lightblue;
}
Background Example

CSS Flexbox

Flexbox helps align items easily. 

.container{

display:flex;

justify-content:center;

gap:10px;

}

CSS Grid

.container{

display:grid;

grid-template-columns:

1fr 1fr 1fr;

gap:15px;

}

CSS Animation

.box{

animation:move 2s infinite;

}

@keyframes move{

from{

transform:translateX(0);

}

to{

transform:translateX(100px);

}

}
